# Chemai

> **语言:** <kbd>[English](README.md)</kbd> <kbd>[**中文**](README.zh.md)</kbd>

**Chemai** 是一个面向 AI 与计算化学交叉方向的项目。当前已经可公开使用的核心内容，是一份精选的双语计算化学开源工具资源清单。

Chemai 的软件部分仍在规划中。现阶段，本仓库主要提供一个结构化参考地图，帮助研究者、学生与工程师快速比较量子化学、分子动力学、原子尺度机器学习、材料工作流、化学信息学、分子对接和可视化方向的开源工具。
