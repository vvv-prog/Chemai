# Contributing

Thank you for helping improve Chemai.

## Resource Collection Updates

For the Awesome Computational Chemistry collection, please include:

- the project website or repository link
- the license declared by the upstream project
- the main programming language or ecosystem
- a concise usage direction
- current release or maintenance status, with a Releases page or changelog link when possible

When adding or changing a resource page, update both the English page and the matching page under
`docs/resources/awesome-computational-chemistry/zh/` when you can.

## Quality Bar

- Prefer open-source projects with clear licenses and real users.
- Keep descriptions factual and concise.
- Avoid promotional wording.
- If update information is time-sensitive, include the month and year.
- Keep internal Markdown links relative and check them before submitting.
